Download the Scarlet Updater

The Australian Armed Forces runs on a purposely built internal software to sync all of our mod repository across everyone who plays with us. It allows us to make small changes in our mod set (including ones we are developing) and to ensure all members have the exact same files at gametime.

  1. When you start up Scarlet, you will be asked for a Scarlet Key. Staff will provide you with the key needed to download the correct modset.
  2. Click Config, and set the directory to the ARMA 3 Directory (Or the directory you want your mods to be copied to).
  3. Click Start Download.
  4. When completed, load up the ARMA Launcher and add the local mod directory "@Mods_AAF".
